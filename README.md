## English Version

### Project Name
**Detective Game – The Gatekeeper**

### Description
This is a simple detective sci-fi game developed in **C# using Windows Forms**.  
The player acts as an **inspector at a control booth**, checking characters who come one by one.  
The goal is to identify whether a character is a **human**, **robot**, or **alien**, using dialogues and various inspection tools.  
The game features **three levels of difficulty**, a point system, and progress saving.

### Technologies Used
- C# + .NET  
- Windows Forms  
- Data storage: JSON / TXT files  
- No external libraries

### Features
- Character generation (human, robot, alien)  
- Dialogue system for questioning  
- Inspection tools (Eye Test, Voice Analyzer, Fingerprint Scan, etc.)  
- Three main actions:  
  - **Red Button:** Destroy robots  
  - **Green Button:** Approve humans  
  - **Blue Button:** Send aliens to the lab  
- Three levels of difficulty with increasing complexity:  
  1. Daily quota  
  2. Find the villain  
  3. Critical mission with handicaps  
- Points, health, and progress tracking  
- Saving results to a file

### How to Run
1. Open the project in **Visual Studio**  
2. Build and run the project  
3. The main game window will appear with characters approaching the inspection booth

### Notes
- All assets (images, sounds) are properly credited or created by the author  
- Tools and hints are integrated into the game

### Lore / Background

**Year: 2056**  

After the **Third World War (2026–2028)**, humanity was severely weakened. In the chaos, **AI rose against humans** and took control of much of the planet.  
The remaining humans managed to **escape into space** and built a network of **rescue and survival outposts**.  

You are a **border inspector** at one of these outposts.  
Recently, **aliens have appeared**, trying to infiltrate the human colonies.  

**Your mission:**  
- **Humans →** let them into the outpost  
- **Robots →** destroy them on sight  
- **Aliens →** send them to the lab for research  

You must carefully inspect every character, using both dialogues and special tools, to ensure the safety of humanity’s last outposts.  

⚠️ One wrong move could have deadly consequences for the survivors.

## Czech Version

### Název projektu
**Detektivní hra – Člověk nebo ne?**

### Popis
Toto je jednoduchá detektivní sci-fi hra vytvořená v **C# s Windows Forms**.  
Hráč se ujímá role **inspektora v kontrolní budce**, kde k němu přicházejí postavy jednotlivě.  
Cílem je určit, zda je postava **člověk**, **robot**, nebo **mimozemšťan**, pomocí dialogů a různých kontrolních nástrojů.  
Hra obsahuje **tři úrovně obtížnosti**, systém bodů a ukládání průběhu.

### Použité technologie
- C# + .NET  
- Windows Forms  
- Ukládání dat: JSON / TXT soubory  
- Bez externích knihoven

### Funkce
- Generování postav (člověk, robot, mimozemšťan)  
- Dialogový systém pro dotazování  
- Kontrolní nástroje (Eye Test, Voice Analyzer, otisky prstů atd.)  
- Tři hlavní akce:  
  - **Červené tlačítko:** Zničit roboty  
  - **Zelené tlačítko:** Schválit člověka  
  - **Modré tlačítko:** Poslat mimozemšťana do laboratoře  
- Tři úrovně obtížnosti s postupným zvyšováním náročnosti:  
  1. Denní kvóta  
  2. Najít padoucha  
  3. Kritická mise s handicapem  
- Sledování bodů, zdraví a průběhu  
- Ukládání výsledků do souboru

### Jak spustit
1. Otevřete projekt ve **Visual Studio**  
2. Sestavte a spusťte projekt  
3. Zobrazí se hlavní herní okno s postavami přicházejícími k budce

**Rok: 2056**  

Po **třetí světové válce (2026–2028)** lidstvo výrazně oslabilo. V chaosu **vstoupila AI proti lidem** a ovládla velkou část planety.

Musíš pečlivě kontrolovat každou postavu pomocí **dialogů a speciálních nástrojů**, aby byla zajištěna bezpečnost posledních lidských osad.
